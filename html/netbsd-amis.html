NetBSD AWS EC2 AMIs

The NetBSD Project provides a number of official AWS EC2 AMIs under https://wiki.netbsd.org/amazon_ec2/amis/.

Unfortunately, at this time, this does not include regularly udpated images for each release. For our class, I have built and provided the following images to use:

NetBSD-current 10.99.2 NetBSD/amd64 t2 us-east-1a ami-05ffda7ac6da57de1 ec2-user
NetBSD 9.1 NetBSD/evbarm-aarch64 a1 and t4g us-east-1a ami-08b87fed21cce91cb root

These AMIs are currently public and should be available to you to launch.
